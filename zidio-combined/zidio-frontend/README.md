# zidio-frontend (Vite + React)

## Quick start
1. `npm install`
2. `npm run dev`
3. Open http://localhost:5173

## Notes
- Update `src/api.js` API_BASE to point at your backend (default: http://localhost:8080).
- Login route expects POST /api/auth/login returning JSON with a token field (e.g. { "token": "..." }).
